## Git branch

### 1. `branch` 생성하기

```
% git branch <branch name>
```

### 2. 새로 생성한 `branch`로 이동하기

```
% git checkout <branch name>
```

### 3. 1번 2번 한번에 하기

```
% git checkout -b <branch name>
```

### 4. `merge` 하기

`merge`를 하는 주체 에서 다음 명령

```
% git merge <object branch>
```
